# AI Career Guidance Chatbot

## Overview

This project is an AI-powered chatbot developed using Python.  
The chatbot collects career-related information through web scraping and uses the collected data as a knowledge source to provide guidance related to internships, careers, and interview preparation.

The main purpose of this project is to create a custom chatbot that helps students get career-related information in an interactive way.

---

## Features

- Web scraping from career guidance websites
- Automatic data collection from online sources
- Knowledge-based chatbot
- Text-based interaction
- Internship guidance
- Career advice
- Interview preparation tips
- Custom dataset creation

---

## Technologies Used

- Python
- Requests
- BeautifulSoup
- Web Scraping
- Natural Language Processing Concepts


---

## How It Works

### Data Collection (Web Scraping)

- The scraper collects career-related information from an online website.
- BeautifulSoup is used to extract useful content.
- The extracted data is stored in `data.txt`.

### Chatbot System

- The chatbot reads the collected information.
- Users can ask career-related questions.
- The chatbot provides guidance based on the available knowledge.
